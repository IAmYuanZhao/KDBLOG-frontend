## Prerequisites

- node &gt;= 6.0.0
- npm &gt;= 3.0.0

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## build for production with minification
```sh
npm run build
```

## Run tests

```sh
npm run test
```
